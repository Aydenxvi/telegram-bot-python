# telegram-bot-python
Телеграм бот для автоматизации ответов-написанный на Python
# 🛍 ТГ-бот для магаза «Лавка Стиля»

Короче, это несложный и шустрый бот для телеги, написанный на **Python** (юзал свежий фреймворк **aiogram 3.x**). Сделал его для бутиков одежды. Бот работает как витрина: клиент заходит, может потыкать каталог по категориям, чекнуть адрес/время работы и, если надо, сразу прыгнуть в ЛС к владельцу.

## 🚀 Что он умеет?

- **Главное меню**: Снизу всегда висят удобные кнопки (`🛍 Каталог`, `📍 Информация`, `📞 Контакты`), чтобы юзер не потерялся.
- **Интерактивный каталог**: Когда жмешь на каталог, бот присылает кнопки прямо под сообщением (инлайн-кнопки). Можно выбрать вещи для парней, девчонок, сумки или аксессуары. Текст меняется прямо в том же сообщении, чат не забивается спамом, и есть кнопка «Назад».
- **Асинхронный движок**: Написан на `asyncio` и роутерах, так что бот не будет тупить, даже если ему начнут писать несколько человек одновременно.
- **Перехват сообщений**: Если клиент напишет боту просто какой-то текст (типа "привет, есть скидки?"), бот вежливо ответит заглушкой, что скоро подключится живой менеджер.

## 🛠 На чем написано?

- **Язык**: Python 3.10+
- **Основная библиотека**: aiogram 3.x (самый топ для ТГ-ботов сейчас)
- **Изоляция**: Виртуальное окружение `.venv`

## 🏁 Как запустить эту тему у себя?

Ничего сложного, ща по шагам разберем:

### 1. Подготовка
Ставь Python (желательно версии 3.10 или новее). Потом иди в телегу к [@BotFather](https://t.me/BotFather) и создавай нового бота через `/newbot`. Забери у него длинный токен (это типа пароля от бота).

### 2. Заходим в папку
Открывай свой терминал и прыгай в папку с проектом:
```bash
cd html.VS
###3. Врубаем виртуалку
Надо создать изолированную папку, чтобы библиотеки не конфликтовали с другими проектами:

Bash
# Создаем окружение
python -m venv .venv

# Включаем на Windows (в VS Code):
.venv\Scripts\activate

# Если ты на Макбуке или Линкусе:
source .venv/bin/activate
4. Ставим библиотеку
Качаем нужный нам aiogram:

Bash
pip install aiogram
5. Настройка токена
Открывай файл bot.py в VS Code, ищи в самом начале строчку BOT_TOKEN = "..." и вставляй туда тот самый секретный токен, который тебе дал BotFather.

6. Старт!
Запускай скрипт командой:

Bash
python bot.py
В консоли высветится Bot is running... — значит, всё четко, бот в сети. Заходи в телегу, находи своего бота и жми «Старт».

Пацаны, ни в коем случае не заливайте свой реальный BOT_TOKEN на открытый GitHub! Если его украдут, чужие челики смогут спамить от лица вашего бота. Для серьезных тем юзайте .env файлы.
